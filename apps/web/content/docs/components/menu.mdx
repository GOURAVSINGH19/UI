## Overview

`DocsSidebarNav` is the vertical menu that powers this documentation site. It
collapses sections, highlights the active page based on the current URL, and
displays contextual badges like “new” or “updated” without extra wiring.

## Installation

Copy `apps/web/components/Sidenav.tsx` into your project or use the CLI:

```bash
pnpm dlx @workspace/cli add menu
```

## When to use

- Documentation or knowledge-base layouts with nested sections
- Desks and dashboards that need a labeled navigation rail
- Marketing launchpads that call out “new”, “updated”, or “beta” screens

## Usage

```tsx
import { DocsSidebarNav } from "@/components/Sidenav";

const sidebar = [
  {
    title: "Components",
    items: [
      { title: "Buttons", href: "/docs/components/button" },
      { title: "Input", href: "/docs/components/input", label: "updated" },
    ],
  },
];

export function Sidebar() {
  return <DocsSidebarNav items={sidebar} />;
}
```

## Accessibility

- Links include focus states and rely on semantic `<a>` tags for screen-reader support.
- Use descriptive titles for every nav item so the sidebar remains scannable.
- Keep badge labels short (≤6 characters) to avoid truncation on narrow viewports.
