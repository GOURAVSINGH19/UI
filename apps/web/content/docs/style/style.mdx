## Global Styles

Add these styles to your `style.css` file before using components. So you can modify the styles as per your need.

```css
:root {
  --foreground: #121212;
  --bg-dark: oklch(0.1 0 264);
  --bg-light: oklch(0.3 0 264);
  --bg: oklch(0.2 0 264);
  --shadow:
    inset 0 -0.01px 0.6px #ffff, 0 1px 2px #00000030, 0 1px 2px 2px #00000030,
    0 2px 2px #00000015;
  --shadow-s:
    inset 0 1px 2px #ffffff30, 0 1px 2px 2px #00000030, 0 2px 2px #00000015;
  --shadow-m:
    inset 0 1px 2px #ffffff50, 0 1px 2px 2px #00000030, 0 2px 2px #00000015;
  --shadow-l:
    inset 0 1px 2px #ffffff70, 0 1px 2px 2px #00000030, 0 2px 2px #00000015;
  --colora: #ffffff;
}
```
